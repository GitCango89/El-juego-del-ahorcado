---

Descripción del Proyecto

Este proyecto consiste en el desarrollo de un juego del ahorcado en Python enfocado en las nuevas tecnologías y su impacto en la sociedad.  

El programa permite al usuario adivinar el nombre de diferentes tecnologías del futuro. Al finalizar el juego, se muestra una reflexión sobre el impacto social de la tecnología correspondiente.

El objetivo principal es integrar los conocimientos adquiridos durante las 8 semanas del curso, aplicando estructuras fundamentales de programación.

---

Objetivo del Programa

Visualizar el impacto de las nuevas tecnologías en la sociedad mediante un juego interactivo que permita reflexionar sobre su influencia en el futuro.

---

Contenidos Aplicados (Integración de las 4 Unidades)

Unidad 1 – Resolución de Problemas
- Análisis del problema
- Diseño del algoritmo
- Diagrama de flujo

Unidad 2 – Manejo de Datos
- Variables
- Operadores
- Entrada y salida de datos

Unidad 3 – Lógica de Programación
- Condicionales (if, else)
- Bucles (while y for)

Unidad 4 – Estructura de Datos y Funciones
- Listas
- Diccionarios
- Funciones
- Uso de la librería `random`

---

Funcionalidades del Programa

- Selección aleatoria de una tecnología.
- Visualización progresiva de la palabra.
- Validación de letras ingresadas.
- Control de intentos.
- Mensaje final con reflexión sobre el impacto social.
- Uso de estructuras repetitivas y condicionales.

---
Cómo Ejecutar el Programa

1. Tener Python instalado.
2. Descargar o clonar el repositorio.
3. Abrir el archivo `main.py`.
4. Ejecutar el programa desde la terminal o desde el entorno de desarrollo.

---

Conclusión

Las nuevas tecnologías tienen un impacto significativo en la sociedad. A través de este proyecto se demuestra cómo la programación puede utilizarse como herramienta para generar aprendizaje y reflexión sobre el futuro tecnológico.

---

Limitaciones

- El programa funciona en consola (modo texto).
- La cantidad de tecnologías es limitada.
- No guarda resultados en archivos externos.
